# 代币领取系统使用指南

## 概述

代币领取系统允许用户通过验证码安全地领取代币奖励。验证码由管理员或系统生成，确保只有授权用户能够领取。

## 功能特性

### ✅ 安全验证
- 通过6位验证码进行身份验证
- 每个验证码只能使用一次
- 验证码有效期：24小时

### ✅ 快速领取
- 验证通过后立即到账
- 无需等待审核
- 链上透明可查

### ✅ 历史记录
- 实时查看领取历史
- 记录验证码、金额、时间
- 链上数据公开透明

## 使用流程

### 1. 连接钱包
- 点击右上角"连接钱包"按钮
- 授权MetaMask访问账户
- 确保钱包地址正确

### 2. 输入验证码
- 在输入框中输入6位验证码
- 验证码必须是大写字母或数字
- 示例：`ABC123` 或 `XYZ789`

### 3. 领取代币
- 点击"立即领取"按钮
- 等待交易确认
- 代币将立即到账

## 验证码规则

### 格式要求
- **长度**：固定6位
- **字符**：大写字母(A-Z)和数字(0-9)
- **不区分大小写**：输入时自动转换为大写

### 使用限制
- 每个验证码只能使用一次
- 使用后立即失效
- 有效期为24小时
- 过期后无法使用

### 验证码获取
验证码由以下方式生成：
- 管理员手动生成
- 系统自动生成（如抽奖中奖、推广奖励等）
- 用户可以通过官方渠道获取

## 合约接口

### 核心函数

#### `claimWithCode(string _code)`
用户使用验证码领取代币

**参数：**
- `_code`: 6位验证码

**示例：**
```javascript
await contract.methods
    .claimWithCode("ABC123")
    .send({ from: userAddress });
```

#### `generateClaimCode(address _user, uint256 _amount, string _code)`
管理员为用户生成领取验证码

**参数：**
- `_user`: 用户地址
- `_amount`: 代币数量
- `_code`: 6位验证码

#### `getUserClaims(address _user)`
获取用户的领取记录

**参数：**
- `_user`: 用户地址

**返回：**
- 用户的所有领取记录数组

#### `getContractStats()`
获取合约统计数据

**返回：**
- `_totalClaims`: 总领取次数
- `_totalClaimed`: 总领取金额

## 事件

### `Claimed`
触发条件：用户成功领取代币

**参数：**
- `user`: 用户地址
- `amount`: 领取数量
- `code`: 使用的验证码
- `timestamp`: 领取时间

### `ClaimCodeGenerated`
触发条件：验证码被生成

**参数：**
- `user`: 用户地址
- `amount`: 代币数量
- `code`: 生成的验证码

## 常见问题

### Q1: 验证码格式是什么？
A: 验证码必须是6位大写字母或数字，例如：`ABC123`、`XYZ789`。

### Q2: 验证码可以使用多次吗？
A: 不行，每个验证码只能使用一次，使用后立即失效。

### Q3: 验证码有效期是多久？
A: 验证码有效期为24小时，过期后将无法使用。

### Q4: 领取代币需要Gas费用吗？
A: 是的，需要支付区块链交易的Gas费用，但不需要额外费用。

### Q5: 如果验证码错误怎么办？
A: 请检查验证码是否正确，确保是6位大写字母或数字，且未过期。

### Q6: 如何获取验证码？
A: 验证码由管理员生成或系统自动生成，请通过官方渠道获取。

### Q7: 领取失败可能的原因？
A: 可能的原因包括：
- 验证码格式错误
- 验证码已使用
- 验证码已过期
- 验证码不存在
- 钱包余额不足（Gas费）

## 安全建议

1. **保护验证码**：不要将验证码分享给他人
2. **及时使用**：验证码应在有效期内尽快使用
3. **确认地址**：领取前确认钱包地址正确
4. **检查网络**：确保连接到正确的网络
5. **记录交易**：保存交易哈希以备查询
