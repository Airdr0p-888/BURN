# 麻了 - 燃烧生态与公平机制网站

## 项目简介
这是一个基于区块链燃烧机制的介绍网站，展示了透明燃烧体系和公平的游戏规则。

## 功能特性

### 🎯 核心板块
1. **Hero 区域** - 醒目的首页展示，突出"双倍燃烧出局"和"币安随机算法助力"
2. **核心机制** - 展示三大核心内容：双倍出局、十代推广、福利奖池
3. **奖池规则** - 详细的奖池参与条件、奖励分配和算法说明
4. **推广奖励** - 完整的十代推广奖励表格
5. **奖池说明** - 资金分配逻辑和实际案例

### 🎨 设计特点
- 现代化渐变配色方案
- 响应式设计，适配各种设备
- 流畅的动画效果和交互体验
- 玻璃拟态（Glassmorphism）设计风格
- 卡片悬停效果和滚动动画

## 技术栈
- **HTML5** - 语义化标记
- **CSS3** - 现代化样式和动画
- **JavaScript** - 交互效果和动画控制

## 使用方法

### 本地预览
1. 确保所有文件在同一目录下：
   - `index.html` - 主页面
   - `styles.css` - 样式文件
   - `script.js` - JavaScript文件

2. 使用浏览器打开 `index.html` 文件

### 服务器部署
```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 使用 Node.js 的 http-server
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 主要内容

### 机制概述
- **永久燃烧**：买卖滑点各3%，全部反哺燃烧，2倍出局
- **机制公平**：合约执行，数据透明，全程可查，币安背书
- **资金闭环**：合理化分配所有资金流向，三重燃烧，双向受利
- **多元福利**：超级奖池+加速燃烧+10代推广等多重福利机制

### 奖池参与条件
- 单笔销毁100U代币 · 抽奖10次
- 单笔销毁500U代币 · 抽奖50次
- 单笔销毁1000U代币 · 抽奖100次
- 额外抽奖通道：单独销毁2U代币，额外获得1次抽奖机会

### 推广奖励
- 直推：5%
- 二代：3%
- 三代：2%
- 四代：1.5%
- 五代：1.2%
- 六至十代：0.9%/0.6%/0.4%/0.3%/0.1%
